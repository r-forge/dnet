Illumina HumanOmniExpress-12 v1.0 annotation data

Description

This dataset contains annotation data for the Illumina HumanOmniExpress-12 v1.0 BeadChip array platform for genome-wide association studies.

Usage

IlluminaOmniExpress <- dRDataLoader(RData = "IlluminaOmniExpress")

Value

a list with 2 components:

  • Info: a data frame
  • GR: an object of class "GRanges"

Examples

IlluminaOmniExpress <- dRDataLoader(RData='IlluminaOmniExpress')
