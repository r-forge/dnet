Function to find the tip node(s) of a direct acyclic graph (DAG)

Description

dDAGtip is supposed to find the tip node(s) of a direct acyclic graph (DAG; an ontology). It return the name (i.e Term ID) of the tip node(s).

Usage

dDAGtip(g)

Arguments

g
an object of class "igraph" or "graphNEL"

Value

  • tip: the tip name (i.e. Term ID)

Note

none

# 1) load HPPA as igraph object
ig.HPPA <-dRDataLoader(RData='ig.HPPA')


'ig.HPPA' (from package 'dnet' version 1.0.7) has been loaded into the working environment

g <- ig.HPPA

tips <- dDAGtip(g)
